### 🗡️ Sızma Testi Uzmanlık Alanları | Penetration Testing Services

Sektörel standartlar (OWASP, PTES, NIST) çerçevesinde aşağıdaki alanlarda profesyonel seviyede sızma testleri ve güvenlik denetimleri gerçekleştiriyorum:

*   🌐 **Web Application Penetration Testing:** Gelişmiş web mimarilerinde iş mantığı hataları (Business Logic Flaws), API zafiyetleri ve OWASP Top 10 standartlarında derinlemesine zafiyet analizleri.
*   📱 **Mobile Application Penetration Testing:** Android ve iOS platformlarında tersine mühendislik (Reverse Engineering), dinamik/statik analiz ve güvenli veri depolama değerlendirmeleri.
*   🏢 **Active Directory & Internal Network Pentest:** Kurum içi (Internal) ağlarda yanal hareket (Lateral Movement), yetki yükseltme (Privilege Escalation), Kerberos tabanlı saldırılar ve Domain Controller ele geçirme senaryoları.
*   🌍 **External Network Pentest:** Kurumun internete açık yüzündeki (Perimeter) zafiyetlerin, açık port servislerinin ve yapılandırma hatalarının tespiti.
*   📶 **Wireless Network Pentest:** WPA2/WPA3 kurumsal ağların güvenliği, Rogue AP saldırıları, istemci izolasyon testleri ve kablosuz ağ bypass teknikleri.
